## 📊 Project Overview
This project performs **Exploratory Data Analysis (EDA)** and applies
**Simple & Multiple Linear Regression** on a COVID-19 health dataset
to predict the number of deaths based on confirmed cases and other features.

---

## 🎯 Objectives
- Perform EDA on COVID-19 dataset
- Visualize patterns and trends in the data
- Build a Simple Linear Regression model
- Build a Multiple Linear Regression model
- Compare both models using evaluation metrics

---

## 📁 Dataset
- **Name:** Corona Virus Report
- **Source:** [Kaggle - imdevskp](https://www.kaggle.com/datasets/imdevskp/corona-virus-report)
- **Type:** Health Dataset
- **File:** covid 19.csv

## 🔍 EDA Steps Performed
1. Dataset loading and shape inspection
2. Data types and basic statistics
3. Missing values analysis
4. Distribution plots
5. Top 10 countries comparison
6. Scatter plots
7. Correlation Heatmap

---

## 🤖 Machine Learning Models

### Simple Linear Regression
- **Feature:** Confirmed Cases
- **Target:** Deaths

### Multiple Linear Regression
- **Features:** Confirmed, Recovered, Death Rate, Recovery Rate
- **Target:** Deaths

---

## 📈 Evaluation Metrics
| Metric | Description |
|--------|-------------|
| R² Score | Higher is better |
| RMSE | Lower is better |
| MAE | Lower is better |
